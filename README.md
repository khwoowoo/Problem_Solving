### 슈도코드 작성 예시 🔥
 - 제일 좋은 방법은 본인이 익숙하고 편한걸로 작성하는 것이 중요!
 - 아래는 DFS 수도코드 예시
```
n(노드 개수) m(에지 개수)
A(그래프 데이터 저장 인접 리스트)
visited(방문기록 저장 배열)

for(n의 개수만큼 반복하기){
  A 인접 리스트의 각 ArrayList 초기화하기
}

for(n의 개수만큼 반복하기){
  if (방문하지 않은 노드가 있으면)
    연결 요소 개수 ++
    DFS 실행하기
}

//DFS 구현하기
function DFS{
  if (현재 노드 == 방문 노드) return;

  visited 배열에 현재 노트 방문 기록하기
  현재 노드의 연결 노드 중 방문하지 않은 노드로 DFS 실행하기(재귀 함수 이용)
}

```
